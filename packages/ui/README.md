## @clobbr/ui
## Getting started

1. Clone the repository
```bash
git clone https://github.com/parsecph/clobbr.git
```

2. Navigate to the UI package directory
```bash
cd clobbr/packages/ui
```

3. Start the App
```bash
yarn start
```

----------------------
